# Python SDK

## Installation

```bash
pip install framelane
```

## Authentication

Set your API key as an environment variable:

```bash
export FRAMELANE_API_KEY=fl_live_...
```

Or pass it directly:

```python
from framelane import FrameLane

client = FrameLane(api_key="fl_live_...")
```

## Quick example

```python
from framelane import FrameLane

client = FrameLane()

asset = client.assets.upload("input.mp4")

job = client.jobs.create(
    assets=[asset.id],
    edits=[
        {"type": "trim", "start": 0, "end": 15},
        {"type": "caption", "style": "subtitles"},
    ],
    output={"format": "mp4", "resolution": "1080p"},
)

result = client.jobs.wait(job.id)
print(result.output_url)
```
