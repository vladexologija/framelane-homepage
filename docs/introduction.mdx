## Why agents need their own video pipeline

- **Identity**: A render pipeline gives an agent a production-grade output channel — it can accept briefs, produce cuts, and deliver finished video.
- **Composition**: Agents can chain primitives (trim, caption, overlay, mix) into complex edits without understanding codecs or timelines.
- **Scale**: Submit thousands of concurrent render jobs. FrameLane handles queuing, encoding, and delivery.

## Key capabilities

- Programmatic asset upload and management via REST API
- Composable edit operations (trim, caption, overlay, audio mix, watermark)
- Webhook events for real-time job lifecycle tracking
- Reusable edit templates for consistent output across campaigns
- Python and TypeScript SDKs
- MCP server for AI tool integration
- OpenAPI specification for codegen and tooling
