# Overlays & Graphics

The `overlay` edit type lets you composite images, logos, and graphics onto any frame of the output video.

## Basic overlay

```python
job = client.jobs.create(
    assets=[footage.id],
    edits=[
        {
            "type": "overlay",
            "src": logo_asset.id,
            "position": "bottom-right",
            "opacity": 0.8,
            "start": 0,
            "end": 30,
        }
    ],
    output={"format": "mp4", "resolution": "1080p"},
)
```

## Position options

| Value          | Placement                         |
|----------------|-----------------------------------|
| `top-left`     | Upper-left corner with padding    |
| `top-right`    | Upper-right corner with padding   |
| `bottom-left`  | Lower-left corner with padding    |
| `bottom-right` | Lower-right corner with padding   |
| `center`       | Centered in frame                 |
| Custom         | `{"x": 100, "y": 50}` in pixels  |

## Stacking multiple overlays

Add multiple overlay edits in the array — they are composited in order (last on top).
