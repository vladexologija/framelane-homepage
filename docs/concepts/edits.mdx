# Edits

**Edits** are the atomic operations applied to assets during a render job. They are composable: stack multiple edits in a single job and FrameLane applies them in order.

## Available edit types

| Type        | Description                                             |
|-------------|---------------------------------------------------------|
| `trim`      | Cut to a time range (`start`, `end` in seconds)         |
| `caption`   | Generate or burn-in subtitles (`style`, `language`)     |
| `overlay`   | Position an image/graphic on the video (`src`, `position`, `opacity`) |
| `audio_mix` | Add background audio, duck, normalize levels            |
| `watermark` | Apply visible or forensic watermark                     |
| `concat`    | Concatenate multiple assets sequentially                |
| `transition`| Crossfade, wipe, or cut between concatenated segments   |
| `resize`    | Scale or crop to target resolution / aspect ratio       |

## Composition

Edits are applied in array order. For example, trim first, then caption, then overlay:

```json
[
  { "type": "trim", "start": 10, "end": 40 },
  { "type": "caption", "style": "subtitles", "language": "en" },
  { "type": "overlay", "src": "ast_logo123", "position": "bottom-right" }
]
```
