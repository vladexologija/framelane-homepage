# Templates

A **Template** is a saved list of edits with optional variable slots. Instead of defining edits per job, reference a template by ID and let FrameLane fill in the operations.

## Creating a template

```python
template = client.templates.create(
    name="branded-short",
    edits=[
        {"type": "trim", "start": "{{start}}", "end": "{{end}}"},
        {"type": "caption", "style": "subtitles"},
        {"type": "overlay", "src": "ast_logo123", "position": "bottom-right"},
        {"type": "watermark", "text": "© 2026 Acme"},
    ],
)
```

## Using a template in a job

```python
job = client.jobs.create(
    assets=[asset.id],
    template_id=template.id,
    variables={"start": 0, "end": 30},
    output={"format": "mp4", "resolution": "1080p"},
)
```

Templates ensure consistency across campaigns and reduce payload size for agents submitting many similar jobs.
